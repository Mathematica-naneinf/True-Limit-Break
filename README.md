# True Limit Break

True Limit Break is an addon for Cryptid, which is a Balatro mod.

As the description in the Json file says, this addon is "balanced" around the Madness gameset, which is the least balanced of Cryptid's 3 gamesets, meaning everything starts unlocked, and effects are pretty powerful.

If you run into any issues with this addon, want to show me something interesting/odd/cool you've done/seen while using it, have any other feedback about it, or have any questions (I have plenty of time on my hands, so don't worry about wasting it), you can mention them to me in the "True Limit Break" thread in `other-mod-forum` in the Cryptid Discord, or by DMing me dierectly (@mathematica_14892).

The aforementioned thread in the Cryptid Discord also contains a variety of terrible ideas, most of which probably aren't getting added any time soon, and some random screenshots and videos related to things I've been working on, some from The Prototype (which I no longer have).
