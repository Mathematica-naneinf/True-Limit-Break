# True Limit Break

True Limit Break is an addon for Cryptid, which is a Balatro mod.

As the description in the Json file says, this addon is "balanced" around the Madness gameset, which is the least balanced of Cryptid's 3 gamesets, meaning everything starts unlocked, and effects are pretty powerful.

If you run into any issues with this addon, want to show me something interesting/odd/cool you've done/seen while using it, or have any questions, you can mention them to me in the "True Limit Break" thread in `other-mod-forum` in the Cryptid Discord, or by DMing me dierectly (@mathematica_14892).
